# 🍽️ MesaExpress — Sistema Inteligente de Pedidos para Restaurantes

O **MesaExpress** é um sistema desenvolvido para otimizar o fluxo de pedidos em restaurantes, conectando clientes, cozinha e caixa de forma simples, rápida e eficiente.

Atualmente o projeto está em desenvolvimento, com o **front-end funcional** e integração com back-end em andamento.

---

## 📌 Objetivo

> Criar um sistema completo de gestão de pedidos para restaurantes, permitindo que o cliente faça pedidos diretamente da mesa e acompanhe o fluxo até a finalização no caixa.

O sistema tem como foco:

* Reduzir erros de comunicação
* Aumentar a velocidade no atendimento
* Melhorar a organização interna do restaurante
* Automatizar o fluxo de pedidos

---

## 🔄 Como funciona o fluxo

1. 🧑‍🍳 Cliente acessa o cardápio (via QR Code ou link)
2. 📲 Realiza o pedido informando seus dados
3. 🍳 O pedido chega na **cozinha**
4. ✅ A cozinha finaliza o pedido
5. 💳 O pedido é enviado para o **caixa**
6. 🧾 Finalização e pagamento

---

## 🧠 O que estou desenvolvendo com este projeto

* Construção de um sistema real com foco em **negócio**
* Organização de fluxo entre múltiplos usuários (cliente, cozinha, caixa)
* Integração entre **front-end e back-end**
* Estruturação de banco de dados
* Pensamento em produto escalável (SaaS)

---

## 🛠️ Tecnologias Utilizadas

* **Next.js** – construção do front-end e renderização
* **Prisma** – ORM para manipulação do banco de dados
* **PostgreSQL** – banco de dados relacional
* **React** – base para construção da interface

## 🖥️ Como rodar o projeto localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/wemersonmatheuss/mesaexpress.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd mesaexpress
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Execute o projeto:

   ```bash
   npm run dev
   ```

5. Acesse no navegador:

   ```
   http://localhost:3000
   ```
